# 📊 Data Analytics Project

## 📌 Overview
This project demonstrates an end-to-end data analytics workflow — from data loading and cleaning to visualization and reporting. The goal is to extract meaningful insights from raw data using Python, SQL, and Power BI.

---

## 📂 Dataset
- Dataset loaded using Python (CSV/Excel format)
- Contains structured data for analysis
- Checked missing values and data types

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)  
- SQL (PostgreSQL / MySQL / SQL Server)  
- Power BI  
- Gamma (for presentation)  

---

## 🔄 Project Steps

### 1. Data Loading
- Loaded dataset using Pandas

### 2. EDA (Exploratory Data Analysis)
- Found patterns and trends using graphs

### 3. Data Cleaning
- Removed null values and duplicates
- Fixed data types

### 4. SQL Analysis
- Ran queries for insights (joins, aggregations)

### 5. Dashboard (Power BI)
- Created interactive dashboard with charts and KPIs

### 6. Reporting
- Summarized insights in report

### 7. Presentation
- Created PPT using Gamma
